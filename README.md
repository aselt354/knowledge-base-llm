\# Автоматизация постановки связей между событиями с помощью LLM



Проект посвящён автоматическому определению связей между событиями в экспертных базах знаний с использованием больших языковых моделей (LLM).



Система анализирует пары клинических наблюдений и определяет:

\- тип связи между событиями;

\- направление зависимости;

\- степень совместимости наблюдений.



Поддерживаемые типы связей:

\- положительная связь;

\- отрицательная связь;

\- несовместимость;

\- отсутствие связи.



Для связанных событий дополнительно определяется характер зависимости:

\- всегда наблюдается;

\- может наблюдаться;

\- всегда отсутствует;

\- может отсутствовать.



Проект исследует применение prompt engineering и chain-of-thought reasoning и другие методы борьбы с галлюцинациями в llm для повышения качества semantic relation extraction в медицинской предметной области.
